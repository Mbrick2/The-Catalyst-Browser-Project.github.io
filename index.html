Private by design

The Catalyst browser has been designed from the ground up to provide a privacy-focused experience. The only server that Catalyst connects to is the update notifier, with no private information sent.

Open Source

All Catalyst code is free and open source for anyone to inspect and review. It is all licensed under MIT and anyone can review all of the code.

Modern

Catalyst is based on Electron, which supports almost all features of Google Chrome with peace of mind. Catalyst also has an elegant user interface with no unnecessary distractions.

Fast

Starts in seconds, no matter the hardware. Simply click on the icon and all of your browsing preferrences will appear in an instant.


This is Catalyst.

We're working to make a fully-compatible with all web standards browser for the masses. We always emphasise privacy and security, which is why we are fully open source.
